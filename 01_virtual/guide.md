## Style guide for Python

https://peps.python.org/pep-0008/

### Creating a new virtual environment

```bash
python -m venv env
```

### Activating a virtual environment

```bash
source env/bin/activate
```

### Deactivating a virtual environment

```bash
deactivate
```
